# Ruta recomendada de certificaciones AWS

Esta ruta está pensada para construir un perfil progresivo en Amazon Web Services (AWS), desde una base de fundamentos cloud hasta un nivel avanzado de arquitectura de soluciones, adecuada para perfiles técnicos, formadores y arquitectos de soluciones.

---

## Nivel Foundational

### AWS Certified Cloud Practitioner (CLF-C02)

![AWS Certified Cloud Practitioner](https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_Cloud-Practitioner_badge.png)

**Objetivo**  
Acreditar el conocimiento de los conceptos fundamentales de la computación en la nube y de la plataforma AWS.

**Qué valida**
- Principios básicos de cloud computing  
- Servicios principales de AWS (EC2, S3, RDS, IAM, etc.)  
- Modelo de responsabilidad compartida  
- Seguridad y cumplimiento  
- Facturación, precios y modelos de coste  
- Casos de uso generales de AWS  

**Cuándo realizarla**
- Primer contacto con AWS  
- Perfiles técnicos que no han trabajado aún en entornos cloud  
- Como certificación base antes de avanzar a niveles Associate  

---

## Nivel Associate

### AWS Certified Solutions Architect – Associate (SAA-C03)

![AWS Certified Solutions Architect – Associate](https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_Solutions-Architect_Associate_badge.png)

**Objetivo**  
Demostrar la capacidad de diseñar arquitecturas escalables, seguras, resilientes y eficientes en coste sobre AWS.

**Qué valida**
- Diseño de arquitecturas altamente disponibles  
- Selección adecuada de servicios AWS según requisitos técnicos  
- Seguridad, redes y control de accesos  
- Optimización de costes y rendimiento  
- Patrones de arquitectura cloud  
- Diseño de soluciones desacopladas y tolerantes a fallos  

**Cuándo realizarla**
- Tras superar Cloud Practitioner  
- Perfiles de desarrollo, sistemas, DevOps o formación técnica  
- Paso clave para roles de arquitecto cloud  

---

## Nivel Associate (opcional, complementario)

### AWS Certified Developer – Associate

![AWS Certified Developer – Associate](https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_Developer_Associate_badge.png)

**Objetivo**  
Validar el conocimiento en desarrollo, despliegue y depuración de aplicaciones basadas en AWS.

**Qué valida**
- Desarrollo de aplicaciones cloud-native  
- Uso de SDKs y APIs de AWS  
- Integración de servicios gestionados  
- Automatización de despliegues  
- Monitorización y troubleshooting  

**Valor añadido**
- Refuerza el perfil de arquitecto con experiencia práctica  
- Muy útil para perfiles con base en desarrollo de software  

---

## Nivel Associate (opcional, complementario)

### AWS Certified SysOps Administrator – Associate

![AWS Certified SysOps Administrator – Associate](https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_SysOps-Administrator_Associate_badge.png)

**Objetivo**  
Acreditar la capacidad de administrar, operar y mantener sistemas en AWS.

**Qué valida**
- Operaciones y mantenimiento de infraestructuras AWS  
- Monitorización, logging y alertas  
- Automatización operativa  
- Gestión de backups y recuperación ante desastres  
- Control de costes y seguridad operacional  

**Valor añadido**
- Complementa la visión de arquitectura con enfoque operativo  
- Ideal para perfiles con background en sistemas o DevOps  

---

## Nivel Professional

### AWS Certified Solutions Architect – Professional

![AWS Certified Solutions Architect – Professional](https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_Solutions-Architect_Professional_badge.png)

**Objetivo**  
Demostrar experiencia avanzada en el diseño de arquitecturas complejas y empresariales sobre AWS.

**Qué valida**
- Diseño de arquitecturas a gran escala  
- Integración de múltiples servicios y cuentas  
- Arquitecturas híbridas y multi-región  
- Estrategias de migración a la nube  
- Optimización avanzada de costes, rendimiento y seguridad  
- Automatización e infraestructura como código  

**Cuándo realizarla**
- Tras experiencia práctica real en AWS  
- Después de Solutions Architect – Associate  
- Para roles de arquitecto senior, consultor cloud o formador avanzado  

---

## Resumen de la ruta recomendada

1. AWS Certified Cloud Practitioner  
2. AWS Certified Solutions Architect – Associate  
3. (Opcional) AWS Certified Developer – Associate y/o SysOps Administrator – Associate  
4. AWS Certified Solutions Architect – Professional
