# Ruta recomendada de certificaciones AWS

Esta ruta está pensada para construir un perfil progresivo en Amazon Web Services (AWS), desde una base de fundamentos cloud hasta un nivel avanzado de arquitectura de soluciones, adecuada para perfiles técnicos, formadores y arquitectos de soluciones.

---

## Nivel Foundational

### AWS Certified Cloud Practitioner (CLF-C02)

<img src="https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_Cloud-Practitioner_badge.png" alt="AWS Certified Cloud Practitioner" width="160"/>

**Objetivo**  
Acreditar el conocimiento de los conceptos fundamentales de la computación en la nube y de la plataforma AWS.

**Qué valida**
- Principios básicos de cloud computing
- Servicios principales de AWS (EC2, S3, RDS, IAM, etc.)
- Modelo de responsabilidad compartida
- Seguridad y cumplimiento
- Facturación, precios y modelos de coste
- Casos de uso generales de AWS

**Cuándo realizarla**
- Primer contacto con AWS
- Perfiles técnicos que no han trabajado aún en entornos cloud
- Como certificación base antes de avanzar a niveles Associate

---

## Nivel Associate

### AWS Certified Solutions Architect – Associate (SAA-C03)

<img src="https://d1.awsstatic.com/training-and-certification/certification-badges/AWS-Certified_Solutions-Architect_Associate_badge.png" alt="AWS Certified Solutions Architect Associate" width="160"/>

**Objetivo**  
Demostrar la capacidad de diseñar arquitecturas escalables, seguras, resilientes y eficientes en coste sobre AWS.

**Qué valida**
- Diseño de arquitecturas altamente disponibles
- Selección adecuada de servicios AWS según requisitos técnicos
- Seguridad, redes y control de accesos
- Optimización de costes y rendimiento
- Patrones de arquitectura cloud
- Diseño de soluciones desacopladas y tolerantes a fallos

**Cuándo realizarla**
- Tras superar Cloud Practitioner
- Perfiles de desarrollo, sistemas, DevOps o formación técnica
- Paso clave para roles de arquitecto cloud
